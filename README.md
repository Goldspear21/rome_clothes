# Rome's clothes
![rome_clothes](https://socialify.git.ci/Goldspear21/rome_clothes/image?forks=1&issues=1&language=1&name=1&owner=1&pulls=1&stargazers=1&theme=Light)

This website is a clothes shopping website, with interactive css design, interactive javascript forms handling, various visually appealing design throughout all pages. The website also has working cart to add clothes to, as well as an order page to enter details, this website was made using HTML5, CSS3, and javascript.
